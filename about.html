About MoneyMinder

MoneyMinder Expense Tracker is a personal project created to help users efficiently track and manage their daily expenses. It provides an easy-to-use interface to add, edit, and delete expenses and visually analyze spending habits through dynamic charts.

This project is built with modern web technologies including HTML, Bootstrap, JavaScript, and Chart.js to deliver a smooth and responsive user experience across devices.
